# Canvas in AI Mode Rolls Out Nationwide: Draft Documents and Build Tools Directly in Search

Google has officially launched Canvas in AI Mode for all U.S. users, transforming how people interact with search results by enabling direct document creation and interactive tool building within the search interface itself.

## What Canvas in AI Mode Does

Canvas in AI Mode eliminates the need to jump between multiple applications when working on projects. Users can now draft documents, create presentations, build calculators, design forms, and develop other interactive tools without leaving their search session.

The feature integrates seamlessly into Google's search environment, allowing users to move from research to creation in a single workflow. Instead of copying information from search results into separate applications, users can immediately apply their findings to tangible outputs.

## Beyond Basic Search Results

Traditional search provides links and snippets of information. Canvas takes this further by offering a collaborative workspace where AI helps transform search insights into actionable materials.

**Key capabilities include:**

- **Document drafting** with real-time AI assistance
- **Interactive tool creation** like calculators and converters  
- **Form building** for surveys and data collection
- **Template generation** for various professional needs
- **Content formatting** and structure optimization

The AI analyzes search context to suggest relevant tools and templates, making the creation process more intuitive and efficient.
